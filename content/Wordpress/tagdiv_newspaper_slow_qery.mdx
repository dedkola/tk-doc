### Fix TagDiv Newspaper Slow Query

Add index to wp_postmeta table to speed up the query when background image is set. And slow query is now fixed.

```sql copy

ALTER TABLE wp_postmeta ADD INDEX meta_key_value_idx (meta_key, meta_value(255));
```
