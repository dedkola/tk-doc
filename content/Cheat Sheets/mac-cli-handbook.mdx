## Terminal Basics

### Opening Terminal

```bash
# Spotlight → type "Terminal" → Enter
# Or: Applications → Utilities → Terminal

# Popular terminal alternatives:
# iTerm2, Warp, Alacritty, Kitty, Hyper
```

### Essential Keyboard Shortcuts

```bash
Ctrl + A            # Move cursor to beginning of line
Ctrl + E            # Move cursor to end of line
Ctrl + U            # Delete entire line (before cursor)
Ctrl + K            # Delete to end of line
Ctrl + W            # Delete word before cursor
Ctrl + R            # Reverse search command history
Ctrl + C            # Cancel current command
Ctrl + D            # Exit shell / EOF
Ctrl + Z            # Suspend current process (resume with fg)
Ctrl + L            # Clear screen (same as clear)
Tab                 # Autocomplete file/directory names
Tab Tab             # Show all possible completions
```

### Command History

```bash
history                         # Show command history
history 20                      # Show last 20 commands
!!                              # Re-run the last command
!n                              # Run command number n from history
!string                         # Run last command starting with "string"
!$                              # Last argument of previous command
Ctrl + R                        # Reverse search through history
fc                              # Open last command in editor
```

## Shell Configuration

### Zsh Config Files (Default Shell)

```bash
~/.zshrc                        # Main zsh config (loaded for interactive shells)
~/.zprofile                     # Login shell config (loaded once at login)
~/.zshenv                       # Always loaded (all zsh invocations)
~/.zsh_history                  # Command history file
~/.zlogout                      # Executed on logout
```

### Bash Config Files (Legacy)

```bash
~/.bash_profile                 # Login shell config
~/.bashrc                       # Interactive non-login shell config
~/.bash_history                 # Command history
~/.bash_logout                  # Executed on logout
```

### Useful Aliases

```bash
# Add to ~/.zshrc
alias ll='ls -la'
alias la='ls -A'
alias l='ls -CF'
alias ..='cd ..'
alias ...='cd ../..'
alias cls='clear'
alias grep='grep --color=auto'
alias rm='rm -i'                # Confirm before deleting
alias cp='cp -i'                # Confirm before overwriting
alias mv='mv -i'                # Confirm before overwriting
alias ports='lsof -i -P -n | grep LISTEN'
alias myip='curl -s ifconfig.me'
alias flushdns='sudo dscacheutil -flushcache; sudo killall -HUP mDNSResponder'
```

### Environment Variables

```bash
echo $PATH                      # Show current PATH
echo $HOME                      # Home directory
echo $SHELL                     # Current shell
echo $USER                      # Current username
printenv                        # List all environment variables
export VAR_NAME="value"         # Set environment variable (current session)
# Add to ~/.zshrc for persistence:
export PATH="$HOME/bin:$PATH"   # Prepend to PATH
```

### Change Default Shell

```bash
chsh -s /bin/zsh                # Switch to zsh
chsh -s /bin/bash               # Switch to bash
cat /etc/shells                 # List available shells
echo $SHELL                     # Show current default shell
```

## Important macOS Paths

### System Directories

```bash
/                               # Root filesystem
/System/                        # macOS system files (read-only, SIP protected)
/Library/                       # System-wide libraries and support files
~/Library/                      # User-specific libraries and app data
/Applications/                  # Installed applications
/Users/                         # User home directories
/usr/local/                     # Homebrew (Intel Mac) and local software
/opt/homebrew/                  # Homebrew (Apple Silicon)
/private/var/                   # Variable data (logs, tmp, etc.)
/private/etc/                   # System configuration files
/tmp/                           # Temporary files (symlink to /private/tmp)
```

### Configuration & Preferences

```bash
~/Library/Preferences/          # App preference files (.plist)
~/Library/Application Support/  # App data and configs
~/Library/Caches/               # App cache files
~/Library/Logs/                 # User app logs
~/Library/LaunchAgents/         # User-level launch agents (auto-start)
/Library/LaunchAgents/          # System-wide launch agents
/Library/LaunchDaemons/         # System-wide launch daemons (root services)
~/Library/Containers/           # Sandboxed app data
~/Library/Group Containers/     # Shared data between apps
```

### Log Files

```bash
/var/log/system.log             # Main system log (legacy)
/var/log/install.log            # Installation logs
/var/log/wifi.log               # Wi-Fi logs
/var/log/apache2/               # Apache web server logs
~/Library/Logs/                 # User application logs
~/Library/Logs/DiagnosticReports/  # Crash reports
# Modern macOS uses unified logging:
log show --last 1h              # Show last hour of logs
log show --predicate 'process == "kernel"' --last 30m  # Kernel logs
log stream                      # Live log stream
```

## File & Directory Management

### Navigation

```bash
pwd                             # Print current directory
cd /path/to/dir                 # Change directory
cd ~                            # Go to home directory
cd -                            # Go to previous directory
cd ..                           # Go up one level
cd ../..                        # Go up two levels
pushd /path/to/dir              # Push directory onto stack and cd
popd                            # Pop directory from stack and cd back
dirs -v                         # Show directory stack
```

### Listing Files

```bash
ls                              # List files
ls -l                           # Long format (permissions, size, date)
ls -la                          # Long format including hidden files
ls -lh                          # Long format with human-readable sizes
ls -lS                          # Sort by file size (largest first)
ls -lt                          # Sort by modification time (newest first)
ls -lR                          # List recursively
ls -1                           # One file per line
ls -d */                        # List only directories
```

### Creating Files & Directories

```bash
touch file.txt                  # Create empty file or update timestamp
mkdir dirname                   # Create directory
mkdir -p path/to/nested/dir     # Create nested directories
mktemp                          # Create a temporary file
mktemp -d                       # Create a temporary directory
```

### Copying, Moving & Renaming

```bash
cp source.txt dest.txt          # Copy file
cp -r source_dir/ dest_dir/     # Copy directory recursively
cp -i source.txt dest.txt       # Interactive (confirm overwrite)
mv oldname.txt newname.txt      # Rename file
mv file.txt /path/to/dest/      # Move file
ditto source/ dest/             # macOS-native copy (preserves metadata)
ditto -V source/ dest/          # Verbose ditto copy
rsync -av source/ dest/         # Sync files (fast incremental copy)
rsync -av --delete src/ dest/   # Sync and delete extra files in dest
```

### Deleting Files & Directories

```bash
rm file.txt                     # Delete file
rm -f file.txt                  # Force delete (no confirmation)
rm -r dirname/                  # Delete directory recursively
rm -rf dirname/                 # Force delete directory
rmdir dirname                   # Delete empty directory only
trash file.txt                  # Move to Trash (if trash CLI installed)
# macOS native: move to Trash via Finder or use `mv file ~/.Trash/`
```

### File Permissions & Ownership

```bash
chmod 755 script.sh             # Set permissions (rwxr-xr-x)
chmod +x script.sh              # Add execute permission
chmod -R 644 directory/         # Set permissions recursively
chown user:group file.txt       # Change file owner and group
chown -R user:group dir/        # Change owner recursively
ls -l                           # View permissions
stat -f '%A %N' file.txt        # Show octal permissions (macOS)
```

### Links

```bash
ln -s /path/to/original link    # Create symbolic link
ln /path/to/original hardlink   # Create hard link
readlink link_name              # Show where symlink points
ls -la                          # Symlinks shown with -> target
```

## Searching & Finding

### Find Files

```bash
find / -name "filename"         # Find file by name (whole system)
find . -name "*.txt"            # Find .txt files in current dir
find . -type f -name "*.log"    # Find files only (not directories)
find . -type d -name "src"      # Find directories only
find . -size +100M              # Find files larger than 100MB
find . -mtime -7                # Files modified in last 7 days
find . -mtime +30               # Files modified more than 30 days ago
find . -empty                   # Find empty files and directories
find . -name "*.tmp" -delete    # Find and delete files
find . -name "*.sh" -exec chmod +x {} \;  # Find and execute command
```

### Spotlight Search (macOS-Specific)

```bash
mdfind "search query"           # Search using Spotlight index
mdfind -name "filename"         # Search by filename
mdfind -onlyin ~/Documents "report"  # Search in specific directory
mdfind "kMDItemKind == 'PDF'"   # Search by file kind
mdls file.txt                   # Show Spotlight metadata for a file
mdutil -s /                     # Check Spotlight indexing status
mdutil -E /                     # Erase and rebuild Spotlight index
```

### Search Inside Files

```bash
grep "pattern" file.txt         # Search for pattern in file
grep -r "pattern" directory/    # Search recursively in directory
grep -i "pattern" file.txt      # Case-insensitive search
grep -n "pattern" file.txt      # Show line numbers
grep -l "pattern" *.txt         # List files containing pattern
grep -c "pattern" file.txt      # Count matching lines
grep -v "pattern" file.txt      # Show lines NOT matching
grep -E "regex" file.txt        # Extended regex (egrep)
```

### Locate Files

```bash
locate filename                 # Fast file search using database
sudo /usr/libexec/locate.updatedb  # Update locate database
which command                   # Show path of a command
whereis command                 # Show binary/man/source paths
type command                    # Show how command would be interpreted
```

## Text Processing

### Common Tools

```bash
sort file.txt                   # Sort lines alphabetically
sort -n file.txt                # Sort numerically
sort -r file.txt                # Reverse sort
sort -u file.txt                # Sort and remove duplicates
uniq file.txt                   # Remove adjacent duplicate lines
uniq -c file.txt                # Count occurrences of each line
cut -d',' -f1,3 data.csv       # Extract columns 1 and 3 (comma delimited)
cut -c1-10 file.txt             # Extract first 10 characters per line
paste file1.txt file2.txt       # Merge lines of files side-by-side
tr 'a-z' 'A-Z' < file.txt      # Convert to uppercase
tr -d '\r' < file.txt           # Remove carriage returns (Windows → Unix)
```

### sed & awk

```bash
sed 's/old/new/' file.txt       # Replace first occurrence per line
sed 's/old/new/g' file.txt      # Replace all occurrences
sed -i '' 's/old/new/g' file    # In-place edit (macOS sed)
sed -n '5,10p' file.txt         # Print lines 5 to 10
awk '{print $1}' file.txt       # Print first column
awk -F',' '{print $2}' data.csv # Print 2nd column (comma delimiter)
awk '{sum+=$1} END {print sum}' file.txt  # Sum first column
```

### Redirection & Pipes

```bash
command > file.txt              # Redirect stdout to file (overwrite)
command >> file.txt             # Append stdout to file
command 2> error.log            # Redirect stderr to file
command &> output.log           # Redirect stdout and stderr
command 2>&1                    # Redirect stderr to stdout
command1 | command2             # Pipe output of cmd1 to cmd2
command | tee file.txt          # Output to screen AND file
command | tee -a file.txt       # Append to file AND show on screen
```

## Process Management

### Viewing Processes

```bash
ps aux                          # All running processes (detailed)
ps aux | grep process_name      # Find a specific process
pgrep -l process_name           # Find process by name (with PID)
top                             # Real-time process viewer
top -o cpu                      # Sort by CPU usage
top -o mem                      # Sort by memory usage
lsof                            # List all open files
lsof -i :8080                   # What is using port 8080
lsof -c process_name            # Files opened by a process
```

### Killing Processes

```bash
kill PID                        # Send TERM signal to process
kill -9 PID                     # Force kill (SIGKILL)
kill -HUP PID                   # Send hangup signal (reload config)
killall process_name            # Kill all processes by name
killall -9 process_name         # Force kill all by name
pkill process_name              # Kill by partial name match
pkill -f "pattern"              # Kill by full command line match
```

### Background & Foreground

```bash
command &                       # Run command in background
jobs                            # List background jobs
fg                              # Bring last job to foreground
fg %1                           # Bring job 1 to foreground
bg %1                           # Resume job 1 in background
nohup command &                 # Run command immune to hangup (persists after logout)
disown %1                       # Detach job from shell
```

### Launch Agents & Daemons (Startup Services)

```bash
launchctl list                          # List all loaded services
launchctl list | grep my-service        # Find a specific service
launchctl load ~/Library/LaunchAgents/com.my.plist    # Load/start a service
launchctl unload ~/Library/LaunchAgents/com.my.plist  # Unload/stop a service
launchctl start com.my.service          # Start a service by label
launchctl stop com.my.service           # Stop a service by label
sudo launchctl kickstart -k system/com.apple.mDNSResponder  # Restart system service
```

```bash
# Keychain
security find-generic-password -a "account" -s "service"  # Find password
security list-keychains         # List keychains

# FileVault (disk encryption)
fdesetup status                 # Check FileVault status
sudo fdesetup enable            # Enable FileVault
```

```bash

```

### Cron Syntax

```bash
# ┌───────── minute (0 - 59)
# │ ┌─────── hour (0 - 23)
# │ │ ┌───── day of month (1 - 31)
# │ │ │ ┌─── month (1 - 12)
# │ │ │ │ ┌─ day of week (0 - 7, 0 and 7 = Sunday)
# │ │ │ │ │
# * * * * * command

# Examples:
0 * * * *     command           # Every hour
*/15 * * * *  command           # Every 15 minutes
0 9 * * 1-5  command            # Weekdays at 9 AM
0 0 * * *    command            # Daily at midnight
0 0 1 * *   command             # First of every month
@reboot      command            # At startup
```

## Useful One-Liners

### File Operations

```bash
# Find and delete .DS_Store files
find . -name ".DS_Store" -delete

# Find large files (>100MB)
find / -type f -size +100M 2>/dev/null

# Count files in directory
find . -type f | wc -l

# Compare two directories
diff -rq dir1/ dir2/

# Generate a file checksum
shasum -a 256 file.txt
md5 file.txt
```

### System Maintenance

```bash
# Free up disk space
brew cleanup && rm -rf ~/Library/Caches/*

# Show top 20 largest files
find / -type f -exec du -h {} + 2>/dev/null | sort -rh | head -20

# List all listening ports with process names
sudo lsof -iTCP -sTCP:LISTEN -n -P

# Kill process on a specific port
lsof -ti :3000 | xargs kill -9

# Watch a command run every 2 seconds
watch -n 2 'kubectl get pods'   # (install via brew)
```

### Text & Data

```bash
# Count lines in all .py files
find . -name "*.py" -exec cat {} + | wc -l

# Remove duplicate lines from a file
sort file.txt | uniq > cleaned.txt

# Generate a random password
openssl rand -base64 24

# Generate UUID
uuidgen

# Encode/decode base64
echo "text" | base64             # Encode
echo "dGV4dAo=" | base64 -d     # Decode

# JSON pretty print
cat file.json | python3 -m json.tool
echo '{"key":"val"}' | jq .     # Using jq (install via brew)
```

## Common Shortnames & Aliases

| Abbreviation | Full Form |
|---|---|
| `~` | Home directory (`/Users/username`) |
| `.` | Current directory |
| `..` | Parent directory |
| `-` | Previous directory (for `cd`) |
| `/` | Root directory |
| `*` | Wildcard (matches any characters) |
| `?` | Wildcard (matches single character) |
| `>` | Redirect output (overwrite) |
| `>>` | Redirect output (append) |
| `|` | Pipe output to another command |
| `&` | Run command in background |
| `&&` | Run next command if previous succeeds |
| `||` | Run next command if previous fails |
| `;` | Run next command regardless |
| `!!` | Repeat last command |
| `!$` | Last argument of previous command |
