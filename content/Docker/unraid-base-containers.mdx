If you're running Unraid at home, these Docker containers can make your server more powerful, flexible, and fun to use! Each container serves a unique purpose, from media streaming to network monitoring and automation. Here’s a must-have list:

- 🗄️ **[Adminer](https://www.adminer.org/):** Database management in a single PHP file.
- 📈 **[Chronograf](https://docs.influxdata.com/chronograf/):** InfluxDB data visualization.
- 🌐 **[Cloudflare Tunnel](https://developers.cloudflare.com/cloudflare-one/connections/connect-apps/):** Securely expose local services.
- 🧲 **[DelugeVPN](https://deluge-torrent.org/):** Torrent client with VPN support.
- 🕸️ **[Flaresolverr](https://github.com/FlareSolverr/FlareSolverr):** Proxy for bypassing Cloudflare protection.
- 📊 **[Grafana](https://grafana.com/):** Analytics & monitoring dashboard.
- 📦 **[InfluxDB](https://www.influxdata.com/):** Time series database.
- 🚦 **[Iperf3](https://iperf.fr/):** Network bandwidth testing.
- 🎬 **[Jellyfin](https://jellyfin.org/):** Media server for streaming.
- 🚀 **[MySpeed](https://github.com/henrywhitaker3/SpeedTestTracker):** Internet speed test automation.
- 🗃️ **[MySQL](https://www.mysql.com/):** Popular relational database.
- 🐘 **[PostgreSQL](https://www.postgresql.org/):** Advanced open-source relational database.
- 🍃 **[MongoDB](https://www.mongodb.com/):** NoSQL document-oriented database.
- 📟 **[Netdata](https://www.netdata.cloud/):** Real-time performance monitoring.
- 📚 **[Next.js Docs](https://nextjs.org/):** Documentation site with Next.js.
- 🗂️ **[phpMyAdmin](https://www.phpmyadmin.net/):** Web interface for MySQL/MariaDB.
- 🎥 **[Plex](https://www.plex.tv/):** Media server for movies & TV.
- 🛠️ **[Portainer](https://www.portainer.io/):** Docker management UI.
- 📡 **[Prometheus](https://prometheus.io/):** Monitoring & alerting toolkit.
- 🦅 **[Prowlarr](https://github.com/Prowlarr/Prowlarr):** Indexer manager for automation.
- 🧲 **[qBittorrent](https://www.qbittorrent.org/):** Lightweight torrent client.
- 📈 **[Telegraf](https://www.influxdata.com/time-series-platform/telegraf/):** Metrics collection agent.
- 📺 **[TVHeadend](https://tvheadend.org/):** TV streaming server.
- ⏰ **[Watchtower](https://containrrr.dev/watchtower/):** Automatic Docker updates.
- 📝 **[WordPress](https://wordpress.org/):** Blogging & website platform.
- 📹 **[youtube-dl](https://youtube-dl.org/):** Download videos from YouTube.
- 📺 **[youtube-dl-server](https://github.com/manbearwiz/youtube-dl-server):** Web UI for youtube-dl.
- 📺 **[youtube-dl-T](https://github.com/yt-dlp/yt-dlp):** Alternative YouTube downloader.
