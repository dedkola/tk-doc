### Install Midnight Commander via Homebrew 

```bash copy
brew install midnight-commander
```

### Fix Permissions Issues

```bash copy

sudo chown -R $(whoami) /Users/$(whoami)/.local
sudo chown -R $(whoami) /Users/$(whoami)/.cache
sudo chown -R $(whoami) /Users/$(whoami)/.config

```
